# 坚果云备份方案

## 背景

语雀曾发生过服务中断事故，为防止数据丢失，将导出的文档备份到坚果云，实现多端同步。

## 方案概述

使用坚果云的「同步指定文件夹」功能，将 `result/` 目录实时同步到云端。文件变动自动上传，其他电脑自动同步下来。

## 当前 result 目录情况

- 总大小：850MB
- MD 文件：540 个
- 图片文件：1847 个
- 坚果云免费版每月上传 1GB，首次同步刚好够用，之后日常增量无压力

## 操作步骤

### 1. 注册坚果云

访问 https://www.jianguoyun.com 注册账号。

### 2. 安装客户端

每台需要同步的电脑都安装坚果云客户端（支持 Windows / macOS / Linux）。

### 3. 设置同步文件夹

1. 安装完成后登录账号
2. 右键 `E:\Y_pythonProject\YuQueToMD\result` 文件夹
3. 选择「Nutstore」→「同步该文件夹」
4. 等待首次同步完成（850MB，取决于网速）

### 4. 其他电脑同步

1. 安装坚果云客户端并登录同一账号
2. 在客户端中找到已同步的 result 文件夹
3. 右键 → 设置同步到本地指定路径
4. 文件自动下载

## 日常使用

### 文档更新后

无需任何操作。以下场景坚果云都会自动同步：

- 重新导出 lakebook 运行转换脚本 → result 目录变化 → 自动上传
- 通过 serve.py 网页端编辑文档 → 文件保存 → 自动上传
- 手动编辑 MD 文件 → 保存 → 自动上传
- 新增图片到 attachments → 自动上传

### 多台电脑协作

- 在 A 电脑改了文档 → 坚果云自动上传 → B 电脑自动同步下来
- 避免同时在两台电脑编辑同一个文件，否则会产生冲突副本

### 冲突处理

如果出现冲突，坚果云会生成带 `(conflicted)` 标记的副本文件，手动对比后保留正确版本即可。

## 免费版限制

| 项目 | 免费版额度 |
|------|-----------|
| 上传流量 | 1GB/月 |
| 下载流量 | 3GB/月 |
| 存储空间 | 无限制（受流量限制） |
| 历史版本 | 1 个月内 |

日常增量更新（改几个文件）远低于 1GB，免费版完全够用。
